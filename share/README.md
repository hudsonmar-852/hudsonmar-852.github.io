# Hudson General Share Guide System

This is Hudson's general mobile HTML guidebook publishing method.

It is not part of the Jeffrey project. Jeffrey is only one possible content topic.

Default repository:

hudsonmar-852/hudsonmar-852.github.io

Default folder:

/share/

Default public URL pattern:

https://hudsonmar-852.github.io/share/[topic-slug].html

## Purpose

Turn raw content into a mobile-friendly HTML guidebook that is easy to read and easy to share.

## Standard Rules

1. Understand the objective, audience, and use case.
2. Improve structure and wording.
3. Create a clear English lowercase file slug from the topic.
4. Save the HTML file under /share/.
5. Page 1 must be the content and objective page.
6. Each page should contain one core idea.
7. Use large Traditional Chinese text and high contrast design.
8. Every page should include bottom navigation:
   - Home to content page
   - Previous page
   - Next page
9. Return the final public GitHub Pages URL.

## Page 1 Should Include

- Objective
- Target reader
- Best use case
- Content structure
- Reading instruction

## Design Style

- Mobile first
- Max width around 430px
- Full screen page sections
- Dark navy or charcoal pages
- Cream text
- Gold, teal, or emerald accents
- Clean and professional

## Command Example

Use the Hudson share mobile guidebook method.

Topic: [topic]
Audience: [reader]
Purpose: [purpose]
Content: [paste content]
Publish under: /share/
