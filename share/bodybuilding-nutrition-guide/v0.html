Core

一句講晒：先食好，再講細節

健美飲食最重要不是神秘食物，而是每日總熱量、蛋白質、碳水、脂肪、睡眠和訓練是否配合。

蛋白質：多數增肌或保肌目標，可用約 1.6–2.2g/kg/day 作參考範圍；備賽很瘦、熱量赤字大時，需求可能更高。
碳水：訓練表現、肌肉飽滿感和恢復很依賴碳水。不是越低碳越專業。
脂肪：不要低到過份，魚、蛋、橄欖油、果仁、牛油果可幫助飲食完整。
最實際：每餐有蛋白質、有蔬菜、有合適碳水；每星期按體重、圍度、訓練表現再微調。

Bulking

Bulking：增肌期點食

增肌期不是放任亂食，而是「有控制地熱量盈餘」。目標是肌肉上升多過脂肪上升。

  • 熱量：慢慢加，不要一開始暴食。
  • 蛋白質：每餐分散攝取，比一次過食晒更易執行。
  • 碳水：放在訓練前後最實用，例如飯、薯仔、燕麥、香蕉。
  • 脂肪：適量，不要長期全靠炸物和加工肉增加熱量。
好 Bulking 應該令你訓練更有力、恢復更好，而不是只令肚腩大得快。

Cutting

Cutting / Firming：減脂期點食

減脂期最怕太急：一急，訓練表現跌、睡眠差、情緒差，肌肉也更難保住。

  • 熱量赤字：逐步減，不建議一刀切。
  • 蛋白質：維持高一點，幫助保肌和飽肚。
  • 碳水：不要亂斬，可集中在訓練日前後。
  • 蔬菜與水分：幫助飽肚、腸胃和日常狀態。
備賽提醒：Peak week、水分、鈉、碳水操作很個人化；本頁只講原理，不提供極端操作流程。

Meal Plan

每日餐單範例

早餐：燕麥 + 希臘乳酪 / 蛋 / 蛋白粉 + 香蕉或莓類。想港式一點：烚蛋 + 麥包 + 無糖豆漿。
午餐：白飯或糙米 + 雞胸 / 魚 / 瘦牛 / 豆腐 + 大量菜。外食可叫少汁、少油、飯量按目標調整。
訓練前：香蕉、飯糰、薯仔、麵包等易消化碳水；不要太油。
晚餐：魚、雞、瘦肉或豆腐 + 菜 + 適量碳水。Cutting 時晚餐不要太鹹，避免水腫感影響判斷。
餐單不是死跟，重點是你知道每餐在提供甚麼：蛋白質、碳水、脂肪、纖維。

HK Eating Out

香港外食選擇

場景較好選擇少啲揀
兩餸飯蒸魚、雞、豆腐、菜、少汁炸豬扒、咕嚕肉、重汁餸
茶餐廳烚蛋、雞扒飯少汁、湯粉走油常餐高牛油、多醬、多加工肉
日式刺身飯、燒魚定食、牛肉飯少汁炸物定食、拉麵加湯飲晒
便利店飯糰、烚蛋、無糖乳酪、低脂奶甜飲、薯片、腸仔、蛋糕
香港外食不是不能健美飲食，而是要學識「蛋白質先行、醬汁分開、炸物少啲」。

Supplements

補充品：邊啲值得

證據較強實用優先

蛋白粉：不是神粉，只是方便補足蛋白質。飲食夠就不是必需。
肌酸 Creatine：對高強度表現、力量和訓練量有較多研究支持；健康成人一般耐受良好。
咖啡因：可提升訓練精神和表現，但太夜用會破壞睡眠。
Omega-3 / 維他命 D / 鎂：視飲食和身體狀況補足，不是越多越好。
不要神化補充品：它們是補位工具，不會取代飲食、睡眠、訓練和健康監測。

Cycle Boundary

如果涉及藥物 cycle

如果有人正在用 AAS 或備賽相關藥物，飲食可以幫助狀態，但不能抵消藥物對心血管、血脂、血壓、肝腎和荷爾蒙的影響。

  • 不要用「多飲水、多食保健品」當保護。
  • 少酒、少炸、少高鈉加工食品，對血壓和血脂更實際。
  • 定期檢查血壓、血脂、肝腎功能和血液指標更重要。
  • 本指南不提供口服藥物、AI、SERM、PCT 或 cycle 安排。
如果主題是比賽表現，飲食要做得精準；如果主題涉及藥物，健康監測要更精準。

Friend Review

如果我是你的朋友，我最想知道……

Q：增肌是不是一定要食到好肥？
不是。好增肌是慢慢加熱量，讓力量和體重穩定上升，而不是靠亂食把脂肪推高。
Q：Cutting 是否越少碳水越快？
短期體重可能跌，但訓練差、睡眠差、狀態扁，未必更好看。
Q：蛋白粉是否必需？
不是。它只是方便。雞蛋、魚、雞、豆腐、奶類、豆類都可以。
Q：最值得先改善甚麼？
先固定每餐蛋白質和每日步數，再調整碳水和總熱量。

Interesting Facts

唔講你唔知

  • 所謂「運動後 30 分鐘黃金窗口」沒有以前講得那麼絕對；全日蛋白質總量和分配通常更重要。
  • 肌酸不是類固醇，它是體內本身存在、亦可從食物取得的化合物。
  • 比賽前看起來「乾」不只是減脂，還涉及肌糖原、鈉、水分、腸胃內容物和皮下水感。
  • 很多人 Cutting 失敗，不是因為不夠狠，而是太早太低熱量，後期沒有空間調整。
最聰明的備賽飲食，不是最極端，而是最可持續、最能保住訓練表現。

3-Min Summary

三分鐘快速看完

  • Bulking 不是亂食，是有控制地熱量盈餘。
  • Cutting 不是越低碳越好，要保住訓練表現和肌肉。
  • 每日蛋白質總量和分配，比迷信黃金窗口更實際。
  • 肌酸、蛋白粉、咖啡因較實用，但補充品不能取代飲食。
  • 香港外食可行：少汁、少炸、蛋白質先行。
  • 如涉及 AAS，飲食不能抵消風險，健康監測才是重點。
